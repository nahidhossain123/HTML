HTML DISPLAY TYPE

Two most common display values are

  1. BLOCK
  2. INLINE

BLOCK

  1. block level element always starts on a new line and browser autometically add spaces(margin) before after it
  2. takes up the full width available

Block level elements are 

<div> <p>

INLINE

  1. does not start on a new line
  2. takes up the as much as width necessary
  3. inline element can not contain block level element

Inline level elements are 

<span> <abbr>
